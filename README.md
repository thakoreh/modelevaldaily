# ModelEval Daily

Daily, repeatable AI model evaluations across coding, reasoning, and tool-use workflows.

## Local dev

```bash
npm install
npm run dev
```

## Deploy (Vercel)

- Framework preset: **Astro**
- Build command: `npm run build`
- Output dir: `dist`

## Notes

- Site URL in `astro.config.mjs` is set to `https://modelevaldaily.com` (update after domain is live).
